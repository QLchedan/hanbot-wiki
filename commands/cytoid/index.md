# Cytoid相关指令
这里是给Cytoid用的一些指令。
<span id="score">

## #score [Cytoid ID] \<NUM>
查询玩家游玩成绩，返回一张图片。

Cytoid ID:必填，玩家的Cytoid名称

NUM:选填，指定查询第几次成绩，不填默认为0。 (0 <= NUM <= 9, 0为最近一次成绩)

&emsp;&emsp;**示例**:
      
&emsp;&emsp;&emsp;&emsp;输入：#score qlchedan

&emsp;&emsp;&emsp;&emsp;输出：![示例](https://files.qlchedan.tk/file/filesssss/hanbot-doc/01.png)
